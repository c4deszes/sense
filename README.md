# Environment sensor

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

> This project implements an environment sensor, including PCB, enclosure and
software.

---

## Features

+ WiFi connection, configuration in AP mode
+ Publishes messages over MQTT
+ Measures several values, including:
  + Temperature
  + Humidity
  + Pressure
  + Dust concentration
  + Wind speed
  + Light level
